SITE ROMÂNTICO TATYANE E JULIA

ARQUIVOS PRINCIPAIS
index.html
style.css
script.js

MÚSICA
Coloque a música desejada em:
assets/audio/musica.mp3

FOTOS
Substitua os arquivos da pasta assets/images mantendo os nomes foto-1.svg até foto-6.svg.
Também é possível usar JPG ou PNG, alterando o caminho correspondente no index.html.

CONTADOR
A data inicial está configurada como 6 de março de 2026 no index.html:
data-start-date="2026-03-06T00:00:00-03:00"

CARTA
A seção da carta está pronta e contém um texto provisório. Quando o texto final for definido, basta substituir o conteúdo dentro de <div class="letter-paper"> no index.html.

COMO ABRIR
Clique duas vezes no arquivo index.html.
Para publicar online, envie toda a pasta para a hospedagem sem alterar a estrutura dos arquivos.
